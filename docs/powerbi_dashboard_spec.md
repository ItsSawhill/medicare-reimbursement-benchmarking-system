# Power BI Dashboard Specification

## Page 1: Executive Overview

Visuals:

- Card: Total Contract Paid
- Card: Total Medicare Benchmark
- Card: Net Dollar Variance
- Card: Avg Contract-to-Medicare Ratio
- Card: Overpayment Count
- Card: Underpayment Count
- Monthly trend line: contract paid, Medicare benchmark, and dollar variance
- Provider risk table: provider, state, risk tier, risk score, recommended action

## Page 2: Provider Benchmarking

Visuals:

- Bar chart: provider ranking by total variance
- Clustered bar: contract-to-Medicare ratio by provider
- Histogram: risk score distribution
- Provider detail table
- Slicers: state, provider type, payer product

## Page 3: HCPCS / Procedure Analysis

Visuals:

- Bar chart: top HCPCS by financial exposure
- Matrix: service category variance
- Column chart: utilization count by procedure
- Stacked bar: over/under benchmark band distribution

## Page 4: Overpayment / Underpayment Review

Visuals:

- Table: overpayment flags
- Table: underpayment flags
- Heatmap or matrix: variance severity by provider/service
- Matrix: benchmark band by provider type and service category

## Page 5: Scenario Modeling & Forecast

Visuals:

- Scenario impact summary table
- Bar chart: rate change impact
- Bar chart: Medicare alignment impact
- Line chart: monthly forecast trend
- Line chart: projected variance trend

## Recommended Slicers

- State
- Provider type
- Payer product
- Service category
- Benchmark band
- Risk tier
- Month
